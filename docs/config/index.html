Configuration Guide

Complete system configuration and setup guide for EyeNet

System Requirements

Minimum Requirements
  • ✓ PHP 8.0+
  • ✓ MySQL 8.0+
  • ✓ 4GB RAM
  • ✓ 2 CPU Cores
Recommended
  • ✓ PHP 8.2+
  • ✓ MySQL 8.0+ with InnoDB
  • ✓ 8GB RAM
  • ✓ 4 CPU Cores

Environment Setup

Environment Variables

# Database Configuration
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=eyenet
DB_USERNAME=root
DB_PASSWORD=secret

# ML Service Configuration
ML_SERVICE_URL=http://localhost:5000
ML_API_KEY=your-api-key

# Network Settings
NETWORK_MONITOR_INTERVAL=60
BANDWIDTH_ALERT_THRESHOLD=0.9

# Security
JWT_SECRET=your-jwt-secret
API_RATE_LIMIT=60
