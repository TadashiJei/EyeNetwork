Predictive Analytics Models

Models Overview

EyeNet employs several predictive models for network analysis and optimization:

  • ARIMA for short-term predictions
  • LSTM for long-term pattern recognition
  • Isolation Forest for anomaly detection
  • Random Forest for classification
